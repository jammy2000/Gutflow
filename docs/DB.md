## Tables

### `price_cache`
15-minute TTL cache for Walmart/Kroger product prices (SSOT §15).

```sql
CREATE TABLE price_cache (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  product_id    TEXT NOT NULL,         -- Walmart itemId or Kroger productId
  store         TEXT NOT NULL,         -- 'walmart' | 'kroger'
  name          TEXT NOT NULL,
  current_price NUMERIC(8,2) NOT NULL,
  original_price NUMERIC(8,2),
  sale_event    TEXT,                  -- 'Rollback' | 'Clearance' | 'Promo' | NULL
  stock_status  TEXT NOT NULL,         -- 'In Stock' | 'Limited' | 'Out of Stock'
  upc           TEXT,
  ingredients_raw TEXT,
  fetched_at    TIMESTAMPTZ NOT NULL DEFAULT now(),
  UNIQUE(product_id, store)
);

-- Auto-purge rows older than 15 minutes (pg_cron or Edge Function)
CREATE INDEX idx_price_cache_fetched ON price_cache(fetched_at DESC);
```

### `meal_plans`

```sql
CREATE TABLE meal_plans (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id         UUID REFERENCES auth.users,

  zip_code        TEXT,
  estimate_mode   BOOLEAN NOT NULL DEFAULT false,
  status          TEXT NOT NULL DEFAULT 'draft',  -- 'draft' | 'confirmed' | 'ordered'
  created_at      TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at      TIMESTAMPTZ NOT NULL DEFAULT now()
);
```

## Price Cache Query Pattern

```typescript
// Fetch with 15-min staleness check (SSOT §15)
const { data } = await supabase
  .from('price_cache')
  .select('*')
  .eq('product_id', productId)
  .eq('store', store)
  .gt('fetched_at', new Date(Date.now() - 15 * 60 * 1000).toISOString())
  .single();

const stale = !data; // cache miss — fetch live
```
