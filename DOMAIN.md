## 2. FODMAP Safety Tier (from SSOT §2.4)

| Tier   | Color | Action             | Stacking Rule                   |
|--------|-------|--------------------|---------------------------------|
| Green  | 🟢    | Allow              | No restriction                  |
| Yellow | 🟡    | Allow + warn       | Synergy multiplier applies      |
| Red    | 🔴    | Block + substitute | Immediate danger, show safe_alt |

**Stacking threshold**: Yellow cumulative load ≥ 10pt → `status = "stacking"`.

Synergy multipliers (same absorption pathway):

| Category        | Multiplier |
|----------------|------------|
| Polyol-Sorbitol | ×1.6       |
| Polyol-Mannitol | ×1.6       |
| Fructan         | ×1.5       |
| Fructan+GOS     | ×1.5       |
| Fructose        | ×1.4       |
| Lactose         | ×1.3       |

## 4. Budget Domain Rules (SSOT §3)

```
subtotal   = Σ ingredient prices (pre-tax)
tax_buffer = subtotal × 0.085
est_total  = subtotal + tax_buffer

Pass condition: subtotal ≤ user_budget
```

## 5. Conflict Resolution Priority (SSOT §8)

1. Medical constraint (FODMAP) > all others
2. User personal exclusions > safe FODMAP ingredients
3. Budget > Nutrition (reduce portions first)
4. Halal + FODMAP must both be satisfied simultaneously
