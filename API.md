## Endpoints

### `POST /api/validate`
**Role**: Medical Brain Gatekeeper — validates any ingredient list.

**Request**
```json
{ "ingredients": ["chicken", "garlic powder", "onion"] }
```

**Response (200)**
```json
{
  "status": "danger",
  "score": 36,
  "grade": "red",
  "reds": [
    { "name": "garlic", "grade": "red", "load": 10, "category": "fructan",
      "reason": "Fructans (high)", "safe_alt": "Garlic-infused oil",
      "fda_position": 2, "fda_weight": 0.667 }
  ],
  "yellows": [],
  "greens": [{ "name": "chicken", "grade": "green", "load": 0, "category": null }],
  "flagged": [],
  "unknowns": [],
  "red_load": 10,
  "yellow_load": 0,
  "category_breakdown": [],
  "portion_items": [],
  "safe_alts": [{ "name": "garlic", "safe_alt": "Garlic-infused oil" }],
  "fda_weights": { "chicken": 1.0, "garlic powder": 0.667, "onion": 0.333 },
  "message": "HIGH RISK: garlic, onion",
  "source": "python"
}
```

**Source field**
| Value | Meaning |
|-------|---------|
| `"python"` | `master_validator.py` answered |
| `"typescript-fallback"` | Python unavailable; TS `validateIngredients()` answered |
| `"empty"` | No ingredients provided |

**GET /api/validate** — health ping, returns `{ "status": "ok" }`.

### `GET /api/market-update`
**Role**: Market Nerve price feed for `page.tsx` polling.

**Response (200)**
```json
{
  "triggers": [
    {
      "item_id": "W001",
      "old_price": 2.98,
      "new_price": 2.48,
      "delta": -0.50,
      "store": "walmart",
      "timestamp": 1708345200000
    }
  ],
  "timestamp": "2026-02-19T12:00:00.000Z",
  "stale": false
}
```

**POST /api/market-update** — engine-room webhook.

**Request**
```json
{ "triggers": [{ "item_id": "W001", "old_price": 2.98, "new_price": 2.48,
                  "delta": -0.50, "store": "walmart", "timestamp": 1708345200 }] }
```
**Response**: `{ "accepted": 1 }`

### `GET /api/lookup-barcode?upc=<string>`
**Role**: UPC → product + raw ingredients for barcode scanner.

**Query params**
| Param | Type | Required |
|-------|------|----------|
| `upc` | string (digits) | ✅ |

**Response (200)**
```json
{
  "upc": "012345678901",
  "name": "Rice Cakes Plain",
  "ingredients": "Rice, Salt",
  "price": 2.48,
  "store": "walmart",
  "in_stock": true
}
```

**Response (404)**
```json
{ "error": "UPC not found", "upc": "000000000000" }
```

## Error Codes (SSOT §14)

| HTTP | `code`                       | Meaning                                      |
|------|------------------------------|----------------------------------------------|
| 400  | `ERR_INVALID_BODY`           | Malformed JSON or missing required field     |
| 400  | `ERR_ZIP_INVALID`            | ZIP code not recognized → switch estimate_mode |
| 404  | `ERR_UPC_NOT_FOUND`          | UPC not in static table or BlueCart          |
| 409  | `ERR_HALAL_FODMAP_CONFLICT`  | Cannot satisfy both simultaneously           |
| 422  | `ERR_BUDGET_TOO_LOW`         | Budget insufficient for minimum plan         |
| 422  | `ERR_NO_SAFE_SUBSTITUTE`     | No FODMAP-safe substitute for OOS item       |
| 504  | `ERR_PRICE_API_TIMEOUT`      | Retailer API unreachable — serve stale cache |
